## Changes This Week
- Migrated to standardized structure


## Token System
I have been using "surface" as my folder to hold color values for visual elements and "text" as the folder for color values of text. I decided on a 4 to 8 point size system for text sizes, as that encompasses most commonly used font sizes.

## Notable changes and decisions
I changed all of the colors and fonts because I decided they were bad! :D 
I intend to continue tweaking the structure of my portfolio until it meets my current satisfactions but that will come as it does. For the most recent changes, I wanted to make this version more space-y than the original. So I fixed the starry background to take up the whole screen and allow elements to scroll up past it. When adding a dark theme, since this version has more space features, I wanted to stay on theme, so I went for a simple black hole-inspired, warm color scheme and I think it's quite fun!
